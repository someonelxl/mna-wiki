# SERIES-001｜马斯克的 Twitter → X → xAI 资本动作链

## 页面定位
本页是一个 Case Series / Case Chain 页面。
它用于串联多个围绕 Twitter / X 平台展开的独立资本动作案例，帮助用户从单次交易视角上升到连续资本动作视角。
本页是组织页，不替代单个 Case 页面。

## 页面职责
本页负责：
- 串联多个独立 Case
- 展示这条资本动作链的顺序
- 说明这些 Case 为什么属于同一条连续动作线
- 帮助用户从整体上理解平台控制权、资产重组与估值逻辑切换之间的关系

本页不负责：
- 替代单个 Case 页面承载完整交易分析
- 重复单个 Case 的全部正文
- 承担规则定义功能
- 承担过重专题分析功能

## 页面基本信息
- `series_id`：`SERIES-001`
- `series_name`：马斯克的 Twitter → X → xAI 资本动作链
- `focus_entities`：Elon Musk / Twitter / X / xAI
- `status`：active
- `scope`：围绕同一平台控制权与后续资产重组展开的连续资本动作线

## 串联对象
本条案例链围绕以下核心对象展开：
- 关键人物：Elon Musk
- 关键平台：Twitter / X
- 关键资产：社交平台资产、AI 资产、平台入口价值
- 串联主题：通过先取得平台控制权，再推动资产重组与叙事切换，重写市场对平台的估值逻辑

## 串联逻辑说明
这条案例链的起点，不只是一次高溢价收购，而是一次围绕平台控制权展开的资本动作。
先收购 Twitter，解决的是控制权与平台归属问题；再推动 X 与 xAI 合并，解决的是资产如何重新组合，以及市场将按什么新逻辑重新定价的问题。
因此，这不是一个单一交易故事，而是一条连续的资本动作链：先控制，再重组，再尝试重写估值逻辑。
