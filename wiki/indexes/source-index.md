# Source Index v0.1

## 页面定位
本页是当前阶段的 Source Record 轻量入口页。
它用于组织和浏览当前正式纳入系统的来源记录对象，帮助用户从知识层回到来源层。
本页是入口页，不替代 Source Record 页面本身。

## 页面职责
本页负责：
- 提供 Source Record 的统一入口
- 提供最小来源识别信息
- 支持从来源记录进入具体 Source Record 页面
- 服务于从来源跳回相关 Case / Concept
- 为后续来源池管理与研究支撑预留位置

本页不负责：
- 代替 Source Record 页面记录来源细节
- 不承担单案例内部来源的精细整理与主组织职责
- 提供 Raw Source 全量浏览入口
- 做复杂来源评分
- 承担规则定义功能

## 当前收录范围
当前阶段，本页收录：
- 已正式建立的 Source Record 页面
- 已具备最小可用状态的来源记录对象
- 标题、来源类型、原始出处关系已基本明确的来源对象

当前阶段暂不收录：
- 仅存在于 raw-sources 但尚未建立 Source Record 的材料
- 临时来源候选
- 还未完成最小记录的来源对象

## 使用说明
建议使用方式：
1. 先看 `source_id`
2. 再看来源标题和来源类型
3. 再看作者/发布方与时间
4. 如需详细信息，请进入对应 Source Record 页面

## 当前列表字段
当前阶段，列表建议至少包含：
- `source_id`
- `title`
- `source_type`
- `author_or_publisher`
- `published_at`
- 可选的主要关联对象
